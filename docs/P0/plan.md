# P0 阶段计划书（增长基础闭环）

## 1. 阶段目标
- 在最小投入下建立用户增长闭环，让博客从“单向发布”变成“可沉淀用户行为”的产品。
- 优先交付高 ROI 能力：用户入驻、基础互动、基础分发、基础留资。

## 2. 业务范围
- 用户体系：注册/登录/基础用户信息。
- 互动体系：评论、点赞（收藏可后置）。
- 分发体系：SEO 基础、Sitemap、RSS。
- 留资体系：订阅入口（邮件/站内订阅）。

## 3. 关键交付
- 用户可完成注册->登录->进入内容消费路径。
- 每篇文章具备基础互动能力并可计量。
- 站点具备基础搜索引擎抓取与订阅分发能力。
- 后台具备基础指标看板（注册数、互动数、订阅数）。

## 4. 指标与验收
- 注册链路成功率 >= 98%。
- 7 日回访率较基线有正向提升。
- 文章互动率（评论/点赞）达到阶段目标。
- 订阅转化率有稳定数据并可复盘。

## 5. 风险与控制
- 风险：功能上线后无有效数据闭环。
- 控制：所有关键行为先埋点、再发布；上线即跟踪转化漏斗。

## 6. 阶段完成定义
- 用户增长闭环稳定运行至少 1 周。
- 关键指标可持续采集并进入周复盘机制。
